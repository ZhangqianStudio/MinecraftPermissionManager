### 什么是SuwingsPermissions？
一个对于Minecraft Server GM插件的权限管理工具。
### 名字为什么这么烂？
对于它的名字而言，是由于缺少一定的思维开阔能力暂时无法得出一个很好的高大上的名字所以根据我自己的名字加上的一个组合。
### 它有什么用？
可以轻松并且一目了然的查看和管理你的Minecraft服务器的权限组。
### 使用？
如果你想编辑它，请勿直接在本项目中编辑。请自行下载。
